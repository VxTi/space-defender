How to play
Hello beginning user!
Welcome to Defender!
Before pressing Play, one must first know the controls.
First, get on thy spaceship. Once aboard, one can move the ship by either occupying the game controller or a keyboard.
If you're using the game controller, the controls are as followed:
To move the spaceship, use the D-Pad buttons (←↑→↓) To shoot, press the lower right button.
To release an explosion, press the upper right button.
To pause the game, press the middle button.
If you're using the keyboard, the controls are as followed:
To move the spaceship, use the WASD keys.
To shoot, press the Space key.
To release an explosion, press the F key.
To pause the game, press the R key.
